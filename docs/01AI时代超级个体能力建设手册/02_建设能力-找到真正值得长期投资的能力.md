# 第二章 建设能力------找到真正值得长期投资的能力

> **本章回答的问题：**
>
> 第一章回答了"为什么AI时代更值得投资能力"，本章回答：
>
> **到底应该优先建设哪些能力，才能在不断变化的时代保持长期竞争力？**

------------------------------------------------------------------------

## 2.1 能力有很多，但不是所有能力都值得优先建设

进入AI时代以后，知识更新越来越快，AI工具层出不穷。真正的问题已经不是"学什么"，而是"先学什么"。

技能会变化，工具会迭代，而真正能够陪伴一个人长期成长的，是底层能力。

因此，本书不追求列出一张越来越长的能力清单，而是聚焦那些能够跨行业、跨岗位、跨时代迁移的核心能力。

> **真正值得长期投资的，不是某项技能，而是能够不断学习新技能、适应新变化、创造新价值的能力。**

### AI在这一阶段的作用

AI降低了知识获取成本，却没有降低能力建设的重要性。

------------------------------------------------------------------------

## 2.2 为什么本书聚焦四项核心能力

本书选择聚焦四项底层能力：

``` text
认知
↓
判断
↓
执行
↓
复盘
```

它们不是全部能力，而是几乎所有复杂能力共同依赖的底层基础。

-   **认知**：理解世界。
-   **判断**：做出选择。
-   **执行**：把想法变成现实。
-   **复盘**：持续优化成长。

创业、管理、教育、AI应用等复杂能力，本质上都建立在这四项能力之上。

------------------------------------------------------------------------

## 2.3 四项能力分别解决什么问题

### 一、认知------理解世界

回答：

> 世界到底是什么样？

建立知识体系、理解规律、形成认知模型。

------------------------------------------------------------------------

### 二、判断------做出选择

回答：

> 什么更重要？什么更值得做？

分析、比较、取舍，并承担选择责任。

------------------------------------------------------------------------

### 三、执行------把想法变成现实

回答：

> 如何真正把判断落实？

持续行动、解决问题、形成结果。

------------------------------------------------------------------------

### 四、复盘------持续成长

回答：

> 如何让下一次做得更好？

总结经验、修正方法、形成长期成长循环。

------------------------------------------------------------------------

## 2.4 能力之上，还有一层更重要的东西

能力不是全部。

真正决定能力最终走向的，是**价值观**。

价值观不是能力。

它决定：

> **哪些事情值得投入。**

因此，本书采用如下模型：

``` text
价值观（方向）
      │
      ▼
认知 → 判断 → 执行
  ↑            │
  └── 复盘 ────┘
```

价值观负责方向。

能力负责成长。

AI负责放大。

------------------------------------------------------------------------

## 2.5 能力只有被外化，才能创造价值

能力如果始终停留在头脑中，就无法创造价值。

本书把能力进入真实世界的过程称为：

> **能力外化。**

能力外化包括：

-   写作
-   演讲
-   视频
-   图片
-   课程
-   咨询
-   产品
-   AI Agent
-   Prompt
-   AI工作流

它们共同承担一个作用：

> **把能力转化为别人能够理解、使用并获得价值的形式。**

AI时代最大的变化之一，就是极大降低了能力外化的门槛。

AI真正放大的，不只是效率，更是普通人的能力外化能力。

------------------------------------------------------------------------

## 2.6 本书能力模型

``` text
                 价值观
                 （方向）

                    │

        ┌─────────────────┐
        │                 │
      认知 → 判断 → 执行
        ↑             │
        └── 复盘 ─────┘
               │
               ▼
           能力外化
（文章｜视频｜课程｜产品｜Agent｜工作流……）
```

能力不断循环。

能力不断外化。

价值不断创造。

------------------------------------------------------------------------

## 本章训练

### 训练一：能力画像

请分别为自己打分（1～10分）：

-   认知
-   判断
-   执行
-   复盘

找出目前最需要提升的一项。

### 训练二：拆解一次真实经历

任选一次重要决策，分别分析：

-   我的认知是什么？
-   我的判断是什么？
-   我是否真正执行？
-   我复盘了什么？

### 训练三：能力外化

请把你最擅长的一项能力，尝试外化为一种成果：

-   一篇文章
-   一个分享
-   一个课程
-   一个Prompt
-   一个Agent
-   一个工作流

目标不是完美，而是完成第一次能力外化。
