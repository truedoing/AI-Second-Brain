# AI时代超级个体能力建设
## Project Roadmap（项目执行路线图）V1.0

> **一句话目标**
>
> 建设《AI时代超级个体能力建设手册》，并通过持续公开实践，验证其理论，形成内容资产、知识品牌与长期商业价值。

---

# 一、项目定位

《AI时代超级个体能力建设手册》不是一本等待出版的书。

它是整个项目唯一的知识源（Source）。

所有内容、产品、课程、案例、商业化，都由此衍生。

项目采用：

> **Think Once, Publish Everywhere**
>
> **一次深度思考，多平台持续运行。**

---

# 二、项目目标（未来12个月）

未来一年，只聚焦一个目标：

> **以《AI时代超级个体能力建设手册》为核心，建立一套可持续发展的 AI 能力建设体系。**

衡量标准：

✅ 持续输出高质量内容

✅ 完成手册持续迭代

✅ 建立稳定内容矩阵

✅ 建立第一批真实案例

✅ 建立第一批商业化收入

除此之外，不主动扩展新的项目。

---

# 三、核心原则

## 1. 手册优先

所有新的思考，首先进入《能力建设手册》。

手册始终是唯一 Source。

---

## 2. 实践优先

任何理论，都必须经过真实发布验证。

遵循：

```

认知

↓

判断

↓

执行

↓

反馈

↓

复盘

↓

更新手册

```

没有实践，不新增理论。

---

## 3. 内容资产优先

优先建设长期资产：

- 能力模型
- 方法论
- 信息图
- 视频
- 案例
- Prompt
- AI工作流

而不是追逐热点。

---

## 4. AI作为能力放大器

AI负责：

- 整理
- 编译
- 优化
- 转换
- 提高效率

人负责：

- 思考
- 判断
- 实践
- 表达
- 价值创造

---

# 四、内容工程（Content Engineering）

整个项目采用软件工程思想管理内容。

```

                深度思考
                    │
                    ▼
        《能力建设手册》（Source）
                    │
                    ▼
              AI Compile
                    │
     ┌────────┼────────┬────────┐
     ▼        ▼        ▼        ▼
 公众号     小红书      B站     抖音/视频号
                    │
                    ▼
              用户反馈（Runtime）
                    │
                    ▼
           Refactor（修改手册）
                    │
                    ▼
              Release V1.1

```

整个项目遵循：

```

Source

↓

Compile

↓

Deploy

↓

Feedback

↓

Refactor

↓

Release

```

不是维护多个平台。

而是维护一个 Source。

---

# 五、一鱼多吃（Create Once, Publish Everywhere）

每完成一个主题。

统一拆解。

```

能力建设手册

↓

公众号长文

↓

信息图

↓

短视频脚本

↓

视频录制

↓

平台文案

↓

多平台发布

↓

收集反馈

↓

修改手册

```

原则：

一次思考。

多次复用。

避免重复创作。

# 六、平台定位

## ① 微信公众号（知识源）

定位：

完整知识沉淀。

发布：

- 系列文章
- 手册章节
- 深度分析

目标：

形成长期知识资产。

---

## ② 小红书（认知传播）

定位：

传播能力模型。

形式：

- 信息图
- 案例
- 观点

目标：

获得精准用户。

验证内容传播能力。

---

## ③ B站（系统表达）

定位：

讲透一个能力模型。

形式：

10~20分钟深度视频。

目标：

建立长期信任。

---

## ④ 抖音 / 视频号（流量入口）

定位：

传播观点。

形式：

30秒~3分钟短视频。

目标：

扩大影响。

不迎合热点。

不改变表达。

---

## ⑤ YouTube（长期国际资产）

定位：

未来全球传播。

现阶段：

同步保存素材。

未来：

增加：

- 英文字幕
- AI配音
- 英文内容

形成国际内容资产。
